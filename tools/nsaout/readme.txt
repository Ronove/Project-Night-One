nsaout v1.2 by alamone.

PURPOSE:
Extract the archive files (*.nsa) from NScripter games.

HOW TO USE:
Run the NSAOUT.EXE in the directory of the game.
The output will be in the "arc/" directory.

SUPPORTED GAMES:

朱 / AKA
みずいろ / Mizuiro
